# Overview

This folder contains concise architecture documentation for the OffroadCamping.Appointments solution.

Contents:

- `architecture-decisions.md` — summary of recorded ADRs and links to full records in `Architecture/ADR`.
- `components.md` — short descriptions of each project and responsibilities.
- `running-locally.md` — quick local run and user-secrets guidance.
- `deployment.md` — deployment notes, migrations and Aspire hints.
- `auth-poc.md` — proposed PoC plan for evaluating Keycloak vs Okta (from ADR 0006).

See the repository README for additional details: [README.md](../README.md)

Use these docs as a quick entrypoint for architects and new contributors.
